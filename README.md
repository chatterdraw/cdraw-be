# ChatterDraw Backend

This repository contains the backend server for ChatterDraw, a real-time collaborative whiteboard.

## Tech Stack
- Node.js
- Express.js
- Socket.IO

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone [your-repo-url]
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the server (dev mode):**
    ```bash
    npm run dev
    ```
The server will be running on `http://localhost:4000`.

## Deployment
This server is configured for automated deployment on [Render.com](https://render.com/). Any push to the `main` branch will trigger a new deployment.
